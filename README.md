# Optimizing Retail Sales and Stock Management for Sahu Cloth House

This project explores sales and inventory data for Sahu Cloth House, a retail store in Haryana, aiming to boost profits and streamline store operations.

## Overview

The analysis identifies top-performing products, uncovers profit leaks, and recommends improvements in inventory and sales management using advanced statistical methods.

## Key Features

- **Weekly Sales Breakdown:** Detailed analysis of product performance using Excel.
- **Pareto (ABC) Analysis:** Focused stock management based on bestselling items.
- **Trend Visualization:** Charts to highlight sales patterns and inform decisions.
- **Actionable Insights:** Data-led recommendations for inventory control and targeted promotions.

## Tools & Methods

- MS Excel (Advanced formulas, pivot tables, charting)
- Statistical analysis (ABC/Pareto, summary statistics, data cleaning)

## Results

- Reduced overstocking and more efficient restocking for popular products
- Better promotional targeting and inventory management
- Practical recommendations for improving cash flow and sales
